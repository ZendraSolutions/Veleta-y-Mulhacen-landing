Teoría de las 3 Capas

Sistema fundamental para mantener el calor corporal en alta montaña

1

Base Técnica

Evacúa el sudor

Camiseta técnica transpirable en contacto directo con la piel. Aleja la humedad del cuerpo.

NUNCA algodón
2

Aislante

Retiene el calor

Plumas o fibra sintética. Crea una cámara de aire caliente.

3

Protección

Corta viento y agua

Impermeable tipo Gore-Tex o similar. Protege de los elementos externos.

Requisitos Técnicos Mínimos

Qué características debe tener tu equipamiento para soportar -10°C en alta montaña

Capa 1: Camiseta

  • ✗ 0% algodón — El algodón se moja y enfría
  • ★ Lana merina — Recomendada (regula temperatura)
  • ✓ Alternativa: sintético (poliéster/poliamida)
  • + Ajustada al cuerpo, cuello alto ideal

Capa 1: Mallas

  • ✗ 0% algodón — Mismo criterio que camiseta
  • ★ Lana merina — Recomendada
  • ✓ Ajustadas, cómodas para caminar

Capa 2: Plumífero

  • ! De montaña, no "de ciudad" — Los finos no sirven
  • ✓ Con capucha — Imprescindible
  • ✓ Función: retener calor — La capa 3 protege del agua
  • + Debe caber sobre la capa 1 sin apretar

Chaqueta Exterior

  • ! Impermeable de verdad — No "resistente al agua"
  • ✓ Costuras selladas — Sin esto, filtra
  • ✓ Cortavientos — Crítico a -10°C
  • + Capucha ajustable y cremalleras de ventilación

Sistema de Guantes

  • ! Un guante normal NO basta a -10°C
  • 1 Guante fino interior — Para manipular cosas
  • 2 Guante/manopla caliente — Para el frío
  • 3 Sobreguante impermeable — Para viento/nieve

Calcetines

  • ✗ 0% algodón — Misma regla que siempre
  • ★ Lana merina — Recomendada (térmica + anti-olor)
  • ✓ Altura media caña mínimo
  • ! No deben apretar (corta circulación = frío)

Botas para Crampones

Crítico
  • ✓ Alta — Cubre bien el tobillo
  • ✓ Caña alta — Botas de trekking con buen soporte de tobillo
  • ✓ Compatible con crampones — Verificar ficha
  • ✓ Impermeable (Gore-Tex o similar)
Talla: +0.5 o +1 número. Probar con calcetín grueso. El talón NO debe bailar al caminar.
